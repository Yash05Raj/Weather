## About

This weather app provides real-time weather data using the OpenMeteo API. Features include:
- Automatic location detection with geolocation
- City search for Indian cities
- Comprehensive weather metrics (temperature, humidity, wind speed, visibility, precipitation, etc.)
- Beautiful glassmorphism UI with responsive design
- Sunrise/sunset times and moon phase information

## Technologies Used

- **HTML5, CSS3, JavaScript (ES6+)** - Vanilla web technologies
- **OpenMeteo API** - Free weather data (no API key required)
